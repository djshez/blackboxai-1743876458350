# Adrianna - Interactive Stories

## Project Overview
Adrianna is an interactive storytelling application where players can choose their own adventure. The game features multiple storylines across various genres, allowing users to explore and shape the narrative based on their decisions. It combines engaging storylines with a sleek, modern interface powered by Tailwind CSS, making it visually appealing and interactive.

## Installation
To run Adrianna locally, follow these steps:

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/yourusername/adrianna.git
   ```
2. Navigate to the project directory:
   ```bash
   cd adrianna
   ```
3. Open the `index.html` or `Adrianna.html` file in your web browser. You can simply double-click the HTML file or use a local server like `live-server` for a better experience.

## Usage
Once the application is up and running, you will be presented with several story options. Click on a story card to begin. As you progress through your chosen story, you will encounter various choices that affect the outcome of the narrative. Make your selections by clicking on the buttons provided to navigate through the story.

## Features
- Multiple interactive storylines in different genres (mystery, romance, adventure).
- Dynamic user interface with smooth animations and transitions.
- Built using HTML, CSS, and JavaScript, with the Tailwind CSS framework for styling.
- Responsive design, ensuring accessibility on various devices (desktop, tablet, mobile).
  
## Dependencies
This project utilizes the following dependencies:
- [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for styling.
- [Font Awesome](https://fontawesome.com/) - Provides vector icons and social logos for the interface.

These are included via CDN links in the HTML files.

## Project Structure
Here is a summary of the project structure for better understanding:

```
/adrianna
│
├── index.html        # Main HTML file for interactive stories
├── Adrianna.html     # Alternate HTML file (duplicate of index.html)
└── /assets           # Future directory for image/video assets (if needed)
```

Inside the `index.html` and `Adrianna.html` files:
- **Header Section**: Displays the title and subtitle of the application.
- **Story Selection Screen**: Contains a grid of cards representing different stories with titles, descriptions, and genres.
- **Story Content Section**: Displays the current story text and choices based on user interactions.
- **JavaScript Logic**: Handles the story progression and user choices through interactive functions.
